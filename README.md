## 📖 Overview

**Fin-AI** is a modern AI-powered financial assistant designed to help users understand, track, and optimize their finances effortlessly.

By combining intelligent insights, financial analytics, and a user-friendly interface, Fin-AI empowers users to make smarter financial decisions and build healthier money habits.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/kunalsrivastava8810/Fin-AI.git
cd Fin-AI
```

### Install Dependencies

```bash
npm install
```

### Start Development Environment

Frontend:

```bash
cd financeMe
npm install
npm start
```

Backend:

```bash
cd backend
npm install
npm run dev
```
